HTML VÀ CSS

Xuất phát điểm là các ghi chú từ các bài giảng về HTML và CSS từ Gà Lại Lập Trình

1 Tổng quan lập trình Web

Danh Dụng Chú thích
Internet mạng lưới toàn cầu kết nối nhiều thiết bị
cáp quang tuyến nối các thiết bị với nhau
Submarine Cable Map xem tuyến cáp quang toàn cầu
clients khách internet là mạng lưới chủ khách
server chủ
IP dãy số xác định vị trí thiết bị trên Internet
DNS (Domain name system) truy cập dùng tên miền thay địa chỉ IP
.html, .css, .js web server phản hồi chúng cho web browser để hiển thị website

2 Sơ sơ website

Danh Dụng Chú thích
Static (tĩnh) website không tương tác với người dùng
Dynamic (động) website phản hồi ontime (kịp thời) với người dùng; facebook, shopee cần các ngôn ngữ backend để sắp xếp và lọc database gửi lên webserver
Frontend phần tiếp cận với người dùng HTML, CSS, JavaSript
Backend phần thực thi các yêu cầu từ frontend các ngôn ngữ lập trình Java, Python, Ruby, PHP; các hệ thống cơ sở dữ liệu MySQL, PostgreSQL, MongoDB

3 Visual Studio Code

Danh Dụng Chú thích
VS Code trình soạn thảo mã nguồn mở, đa năng, miễn phí; Microsoft
File/Open Folder (Ctrl+O) mở thư mục lưu công việc
File/Close Folder đóng thư mục đang hiện hành
Explorer/New File tạo tệp mới
index.html trang chính được trình duyệt hiển thị khi truy cập
! + Enter (Tab) khởi tạo cấu trúc tiêu chuẩn của tệp .html
Manage/Settings/Editor: Font Size tăng kích thước chữ vùng soạn thảo
Shift + Alt + R mở thư mục dự án nơi tệp đang hiện hành chuột giữ tại Explorer
h1 + Tab tạo thẻ <h1></h1>
<h1> HTML và CSS </h1> đánh dấu và định dạng "HTML và CSS" là tiêu đề lớn nhất tiêu đề lớn nhất thường là tiêu đề chính của trang, do đó chỉ nên có một
Internet Archive dự án lưu giữ các website toàn cầu

4 Một số tiện ích

Danh Dụng Chú thích
Prittier tự động định dạng mã nguồn theo một qui ước nhất định, giúp duy trì tính nhất quán của nó
Manage/Settings/Editor: Default Formatter/Prettier - Code formatter xác định trình định dạng mặc định là Prettier
Manage/Settings/Editor: Format On Save tự động định dạng khi lưu
Manage/Settings/Files: Auto Save/onFocusChange tự động lưu khi chuyển sang cửa sổ khác
Manage/Settings/Files: Auto Save/afterDelay tự động lưu sau một khoảng chờ
Material Icon Theme cung cấp các biểu tượng cho các loại tệp và thư mục khác nhau tải về cài, không có sẵn trong Extensions của VS Code
Extensions/Material Icon Theme/Set File Icon Theme/Material Icon Theme thiết lập sử dụng Material Icon Theme
Image preview xem trước các tệp hình ảnh trực tiếp từ trình soạn thảo
Color Highlight hiển thị trực tiếp màu sắc khi đặt con trỏ trên mã màu hoặc mã hex
